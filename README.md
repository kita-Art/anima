# 🚀 도형 로딩 애니메이션 (Shape Loading Animation) - CSS 설명

이 프로젝트는 HTML과 CSS만을 사용하여 간단하면서도 시각적으로 매력적인 도형 로딩 애니메이션을 구현합니다. 세 개의 원형 요소가 순차적으로 커지고 작아지며 사라지는 효과를 통해 사용자에게 로딩 중임을 알립니다.
